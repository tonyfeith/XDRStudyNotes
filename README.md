# XDRStudyNotes

这个仓库用于记录我在无线/通信领域学习 XDR（External Data Record，外部数据记录）指标的过程，包括基础概念、指标含义、分析方法、问题定位和实践复盘。

目前仓库刚刚建立，内容会随着日常学习逐步补充。

## 学习范围

- XDR 指标基础概念与采集位置
- 通信网络中的业务感知与质量指标
- DNS、TCP、TLS/SSL、HTTP 等协议指标分析
- DPI 探针上下方向的问题定界方法
- 无线侧、核心网侧、互联网侧和 SP 侧的实践案例学习
