# XDRStudyNotes 项目约定

## 目标

记录通信领域 XDR 的日常学习内容，形成可检索、可复盘、可持续更新的 Markdown 知识库。

## 文件结构

- `README.md`：仓库说明、学习范围和笔记索引。
- `notes/`：日常学习笔记，仅放 Markdown 文件。
- `notes/README.md`：笔记目录规则和写作模板。

## 命名规则

- 日常笔记使用 `YYYY-MM-DD-topic.md`。
- `topic` 使用简短英文小写单词，单词之间用连字符连接。
- 一个文件聚焦一个主题；主题发生明显变化时新建文件。

## 内容规则

- README 中的笔记索引与 `notes/` 内文件保持同步。
- 引用外部资料时标明来源链接和访问日期。
- 未验证的判断明确标注为“待验证”，不与事实混写。
- 不提交密钥、Token、密码、客户数据、内部网络信息或其他敏感内容。

## 清理规则

- 重复内容合并后删除旧文件。
- 失效链接在复查时更新或标记。
- 临时草稿若 30 天未继续整理，移入对应正式笔记或删除。
